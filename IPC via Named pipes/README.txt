Client-Server Messaging App
----------------------------

STEPS TO RUN:
	1. Compile server.c by running : gcc -o server server.c
	2. Compile client.c by running : gcc -o client client.c
	3. Open two separate terminals
	4. Run ./client in one of the terminals
	5. Run ./server in the other terminal
	6. Messaging can be done between the 2 terminals now
